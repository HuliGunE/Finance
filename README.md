# Доходи та витрати — Android-додаток

Додаток для обліку доходів і витрат по фірмах (організаціях). Інтерфейс українською.

---

## Що робить додаток

- **Додавати фірми** — кожна фірма має назву (наприклад, ТОВ «Ромашка»).
- **В межах кожної фірми** — додавати **доходи** (скільки прийшло грошей) і **витрати** (на що пішли гроші).
- **Бачити залишок** — загальний баланс по фірмі, суму доходів і суму витрат.
- Всі дані зберігаються в телефоні (база даних на пристрої).

---

## Як зібрати APK (покрокова інструкція для новачка)

### Крок 1. Встановити Android Studio

1. Відкрийте в браузері: **https://developer.android.com/studio**
2. Натисніть **Download Android Studio**.
3. Запустіть завантажений файл і пройдіть майстер встановлення (всі кроки можна залишати за замовчуванням).
4. При першому запуску Android Studio зачекайте, поки завантажаться необхідні компоненти (SDK тощо).

### Крок 2. Відкрити проєкт

1. Запустіть **Android Studio**.
2. В меню виберіть **File → Open** (або **Open** на стартовому екрані).
3. Вкажіть папку з цим проєктом:  
   `c:\Users\Jagyar\Desktop\Расходи и доходы`
4. Натисніть **OK**. Почекайте, поки проєкт відкриється і Gradle підтягне залежності (внизу буде прогрес). Це може зайняти кілька хвилин.

### Крок 3. Зібрати APK

1. У верхньому меню виберіть **Build → Build Bundle(s) / APK(s) → Build APK(s)**.
2. Дочекайтесь повідомлення **APK(s) generated successfully**.
3. У спливаючому вікні можна натиснути **locate** — відкриється папка з готовим файлом **app-debug.apk** (зазвичай це щось на кшталт:  
   `Расходи и доходы\app\build\outputs\apk\debug\app-debug.apk`).

### Крок 4. Встановити APK на телефон

1. Скопіюйте **app-debug.apk** на телефон (месенджер, хмара, USB).
2. На телефоні відкрийте файл і дозвольте встановлення з невідомого джерела (якщо система запитає).
3. Після встановлення запустіть додаток **«Доходи та витрати»**.

## Як користуватися додатком

### Головний екран — «Мої фірми»

- Показує список усіх ваших фірм і залишок по кожній.
- **Що натиснути:** кнопку **«+»** (внизу справа) — відкриється екран **«Додати фірму»**.
- Введіть назву фірми (наприклад, «ТОВ Ромашка») і натисніть **«Зберегти»**.
- Щоб відкрити фірму та переглянути доходи/витрати — натисніть по картці цієї фірми.

### Екран фірми — доходи і витрати

- Зверху — **залишок** (баланс), **всього доходів** і **всього витрат**.
- Нижче — список усіх операцій (доходи та витрати).
- **«Додати дохід»** — записати, скільки грошей прийшло; вкажіть суму (грн) і опис (звідки/за що).
- **«Додати витрату»** — записати, на що пішли гроші; вкажіть суму і опис.
- Довге натискання на запис у списку — видалити цей запис (з’явиться підтвердження).

### Повернення назад

- Стрілка **←** в лівому верхньому куті екрана повертає на попередній екран.

## Зв’язок з GitHub (репозиторій HuliGunE/Finance)

Якщо ви хочете зберегти цей код у ваш репозиторій **https://github.com/HuliGunE/Finance**:

1. Встановіть Git: **https://git-scm.com/download/win**
2. В Android Studio: **VCS → Enable Version Control Integration** → виберіть **Git**.
3. Далі: **VCS → Git → Remotes…** → додайте адресу:  
   `https://github.com/HuliGunE/Finance.git`
4. Потім: **VCS → Commit** (зафіксуйте зміни), далі **VCS → Push** (відправка на GitHub).

Або в терміналі (в папці проєкту):

```bash
git init
git add .
git commit -m "First commit"
git remote add origin https://github.com/HuliGunE/Finance.git
git branch -M main
git push -u origin main
```

(Потрібен буде логін і пароль/токен GitHub.)
